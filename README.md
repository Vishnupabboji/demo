# 📝 Simple To-Do List

A beginner-friendly to-do list application built with HTML, CSS, and JavaScript. Perfect for learning Git workflows and web development basics!

## ✨ Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks
- ✅ View task statistics (total & completed count)
- ✅ Data persists using browser localStorage
- ✅ Clean, modern UI with gradient background
- ✅ **Fully responsive design** - works perfectly on mobile & desktop
- ✅ Touch-friendly buttons and interactions
- ✅ **Dark mode toggle** - switch between light and dark themes

## 🚀 How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. Start adding tasks!

### Features:
- **Add Task**: Type in the input field and click "Add" or press Enter
- **Complete Task**: Click the checkbox next to a task to mark it done
- **Delete Task**: Click "Delete" to remove a task
- **Track Progress**: See total and completed task count at the bottom
